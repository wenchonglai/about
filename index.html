Make story-telling interactive maps.

Since 2013, I have started developing web applications for interactive maps using JavaScript, jQuery, SVG, Canvas, and React. Compared to the static maps that planners conventionally produce, these interactive maps were much more powerful in conveying comprehensive, informative, and personalized messages tailored to each and every individual amongst thousands of recipients. These interactive maps were also proven to be more cost-efficient, as the web-based approach eliminates countless redraws that may incur during design iterations.
